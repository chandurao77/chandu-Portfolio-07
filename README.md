# chandu-Portfolio-07

A React-based developer portfolio showcasing projects, skills, and professional experience — built with Create React App.

## Live Demo

[View Portfolio](https://chandumarrupublicportfolio.vercel.app/)

## Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5 / CSS3
- Create React App

## Features

- Responsive design for all screen sizes
- Projects showcase with descriptions and links
- Skills and tech stack overview
- Professional experience timeline
- Contact section

## Getting Started

```bash
git clone https://github.com/chandurao77/chandu-Portfolio-07.git
cd chandu-Portfolio-07
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
